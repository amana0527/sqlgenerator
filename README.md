## Description

The **SQL Generator** project converts natural language questions into SQL queries using Python.
Users can type a question in simple English, and the system automatically generates the corresponding SQL query.

This project is designed to make database querying easier for beginners who may not be familiar with SQL syntax.
The notebook runs in Google Colab and demonstrates how natural language processing concepts can be applied to generate database queries.

**Key Features:**

* Accepts user input in plain English
* Automatically generates SQL queries
* Beginner-friendly implementation using Python
* Runs directly in Google Colab for easy execution
